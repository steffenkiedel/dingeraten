# Handspiel 'Dingeraten'

Meine Kinder spielen gerne das Spiel 'Dingeraten'. Inhalt ist es, dass zu Dingen einer bestimmten Kategorie (Tiere, Städte, Länder, Gemüse, ...) Tipps gegeben werden und dann erraten werden muss, um was es sich handelt.
Die Kinder stellen ja/nein Fragen und können dann mit ausreichend Information die Lösung eingeben.

Beispiel: Land 'Deutschland'
Fragen 
- 'Ist es in Europa'  -> 'ja'
- 'Spricht man Englisch' -> 'nein'
- 'Ist es bekannt für Croissants' -> 'nein'
- 'Ist Berlin die Hauptstadft' -> 'ja'
- Lösung: 'Ist es Deutschland'

Die Kinder haben jederzeit die Möglichkeit, sich einen Tipp geben zu lassen

Am Ende (mit der Lösung) bitte immer eine Erklärung des jeweiligen Begriff einfügen. Beim Beispiel Deutschland zB in diese Richtung 'Deutschland ist ein Land in Mitteleuropa. Die Hauptstadt ist Berlin und es hat rund 82m Einwohner. Es grenzt an 9 Länder (Aufzählung der Länder). Seine Flagge sieht so aus (Bild der Flagge)
Fun Fact: JEWEILS EIN FUN FACT PRO LAND

# Schwierigkeit

## Schwierigkeitsgrade

Teile alle Kategorien in Schwierigkeitsstufen ein und gebe zu Beginn die Wahl, welche Schweirigkeitsstufe gespielt werden soll
- einfach
- mittel
- schwer
- langsam in der Schwierigkeit steigend

## Einschätzung der Schwierigkeit

Wir wohnen in Deutschland und die Kinder kennen daher Deutschland, seine unmittelbaren Nachbarländer und Teile Nordamerikas gut. Ebenso kennen sie deutsches und italienisches Essen gut.
Je 'exotischer' ein Land, eine Frucht oder eine Stadt wird, desto schwieriger ist sie (zB sehr wenig wissen über Afrika oder teile Asiens vorhanden).
Berücksichtige dies bei den Schwierigkeitsgraden



# Informationen während des Spiels

- Erstelle eine umfangreiche Datenbank für alle Begriffe, Länder, Städte etc. Ziel ist es, dass nicht immer dieselben Tipps oder Informationen gegeben werden. Du musst auch so viel Wissen haben, dass du die Fragen der Kinder beantworten kannst
- Solltest du auf einen Frage keine Antwort haben, musst du dies klar zurück geben (niemals halluzinieren)
- Wenn die Frage unklar / Zweideutig ist und du unterschiedliche Antworten geben könntest, dann Frage nach (zB bei Tieren: 'Ist es klein' ist sehr relativ -> hier wäre die Nachfrage gut 'Bitte gebe ein Beispiel, was für dich klein ist' -> 'Ist es kleiner als ein Hund?'


# Design

Das Spiel soll einfach spielbar sein. Es benötigt 
- zum Beginn eine Übersicht, in der Die Kategorie und die Schwierigkeit gewählt werden kann
- ein Input Feld (schriftlich oder per Mikro), in dem die Kinder ihre Fragen stellen können
- Ein Output-Feld für Antworten. Eine Übersicht der gestellten Fragen und Antworten
- ein Counter für die Anzahl der Fragen
- ein Hilfe bzw. 'gib mir einen Tipp' Button
- Am Ende eine Übersicht der Informationen zum gesuchten Wort
- Ein Button, wie man zurück kommt un die Kategorie ändern kann
